 ## 1.2 编写第一个netty客户端与服务器
 >2018年8月23日10:42:27
 ### 1.2.1 服务端
   - 至少一个`ChannelHandler`---- 该组件实现了服务器对
   从客户端接受的数据的处理，即它就是业务逻辑处理器
   - 引导(`ServerBootsStrap`) ---- 这是配置服务器的启动代码，至少
   ，它会将服务器绑定到了它要监听的连接请求的端口。
  ### 1.2.2 客户端
   1、 连接服务端
   
   2、 发送一个或者多个消息
   
   3、 对于每个消息，等待并接受从服务端返回
   
   4、 关闭连接
   
   与服务端相同，一个`ChannelHandler`处理客户端业务逻辑
   ，一个引导客户端(`BootStrap`)。
